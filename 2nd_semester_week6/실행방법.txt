[파일 준비]
1) sendmail.py 와 같은 폴더에 mail_target_list.csv 파일을 만든다.
2) CSV 형식(헤더 포함):
   이름,이메일
   홍길동,abc123@gmail.com
   김철수,cs_kim@naver.com

3) CSV 인코딩은 UTF-8(BOM 포함, utf-8-sig) 권장.
   엑셀에서 깨지면 "UTF-8 BOM"으로 다시 저장한다.

[기본 실행 - Gmail / 한 명씩 반복 발송(권장, 좋은 방법)]
python sendmail.py --from-email 본인계정@gmail.com --subject "제목" --html "<p>HTML 본문</p>"

- send-mode 기본값은 single 이므로 한 명씩 반복 발송된다.
- 비밀번호는 실행 후 프롬프트에 입력하거나,
  환경변수 GMAIL_APP_PASSWORD 로 설정할 수 있다.

[일괄 발송 모드 - Bcc로 한 번에 전송(수신자 주소 숨김)]
python sendmail.py --from-email 본인계정@gmail.com --subject "제목" --html "<p>HTML 본문</p>" --send-mode bulk

- bulk 모드는 Bcc(숨은참조) 방식으로 발송되어
  수신자들이 서로의 이메일 주소를 볼 수 없다.

[텍스트 본문도 함께 보내기]
python sendmail.py --from-email 본인계정@gmail.com --subject "제목" --body "텍스트 본문" --html "<p>HTML 본문</p>"

- 텍스트와 HTML이 함께 있으면 멀티파트로 전송된다.

[첨부파일 포함]
python sendmail.py --from-email 본인계정@gmail.com --subject "제목" --html "<p>HTML 본문</p>" --attachments file1.png file2.pdf

[네이버 SMTP로 발송 시도]
python sendmail.py --from-email 본인계정@naver.com --subject "제목" --html "<p>HTML 본문</p>" --smtp-host smtp.naver.com --smtp-port 587

[수신자 직접 지정(to 옵션)]
python sendmail.py --from-email 본인계정@gmail.com --to test1@gmail.com test2@naver.com --subject "제목" --html "<p>내용</p>"

- --to 를 쓰면 CSV 대신 해당 주소로 발송된다.

[비밀번호 입력 우선순위]
1) --password 옵션
2) 환경변수 GMAIL_APP_PASSWORD
3) 실행 시 프롬프트 입력
