사용 예시:
1) 텍스트 메일
   python sendmail.py \
       --from-email you@gmail.com \
       --to to1@example.com to2@example.com \
       --subject '테스트 메일' \
       --body '안녕하세요. 본문 내용입니다.'

2) HTML 본문
   python sendmail.py \
       --from-email you@gmail.com \
       --to to@example.com \
       --subject 'HTML 테스트' \
       --html '<h1>안녕하세요</h1><p>HTML 본문</p>'

3) 첨부파일
   python sendmail.py \
       --from-email you@gmail.com \
       --to to@example.com \
       --subject '첨부 테스트' \
       --body '첨부 확인 바랍니다.' \
       --attachments ./report.pdf ./image.png

비밀번호 입력 우선순위:
--password 인자 > 환경변수 GMAIL_APP_PASSWORD > 실행 중 프롬프트 입력(getpass)
